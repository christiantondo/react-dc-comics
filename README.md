### NOTE
Ricordate che stiamo lavorando con React: per vedere qualcosa ho bisogno di lanciare npm run dev e puntare il browser su localhost. Chiuso il terminale, il server smette di funzionare. ⚠️
 Non possiamo usare l'estensione live-server né aprire l'index.html direttamente in browser.
E' un esercizio in più fasi. La parte del "main" per ora sarà un fascione nero con un segnaposto "content qui". Verrà sviluppata nelle fasi successive.
